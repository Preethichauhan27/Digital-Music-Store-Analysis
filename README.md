# 🎵 Music Store SQL Analysis

## 📌 Project Overview

Analyzed a music store database using PostgreSQL to solve business-focused analytical problems and generate meaningful insights from sales and customer data.

---

## 🛠️ Tools Used

- PostgreSQL
- pgAdmin 4
  
---

## 📚 SQL Concepts Covered

- Basic SQL Queries
- Joins
- Aggregate Functions
- GROUP BY & HAVING
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
 
---

## 📊 Project Highlights

- Solved **business-oriented SQL problems** from basic to advanced level.
- Performed customer, sales, revenue, and product analysis.
- Applied SQL techniques to extract business insights and support data-driven decision-making.
